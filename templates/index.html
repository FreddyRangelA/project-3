What is sentiment analysis?






were we correct?

Sentiment analysis is the use of natural language in order to process: text analysis, computational linguistics, and biometrics to systematically identify, extract, quantify, and study subjects' information from simple internet’s comments.

Sentimental analysis allows us to collect data from thousands of comments in regards to one topic and analyze the overall inclination towards one side or the other, to understand whether the overall sentiment is negative, positive or neutral.

Data and data colection

The internet is full of opinions, some of them are constructive, some of them not so much. Therefore one good place to encounter as many and as politicized as possible was reddit.
Read more about PSAW and PushshiftAPI
In order to collect the desired data, we used the PSAW reddit wrapper collected by PushshiftAPI that lets us get historical Reddit data updated often was imported and implemented. This wrapper allows users to quickly scrap the needed data from reddit comments with just a few simple parameters.


From psaw import PushshiftAPI. This parameter allows searching on the API for a specific parameter. This parameter would be define under “q” for the main query, “subreddit” for more accuracy and finally “filter”

Next, the data is placed on a Pandas DataFrame.

The data went under minor clean up. Several columns were dropped, time column was modified to show the correct time format and the data selection based on time was made from 2015 to 2021.
Finally, the data can be saved on a CSV file on any given path. After this, more modifications can be made. A column named “Sentiment” was created and given the values 1= positive posts, 2= negative posts, 0= null posts and 3= posts needed to be removed.
